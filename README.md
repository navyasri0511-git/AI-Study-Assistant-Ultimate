# 🚀 AI Productivity Suite

An AI-powered productivity platform built using Python and Streamlit that helps users analyze datasets, evaluate resumes, generate code, and interact with an AI assistant.

---

## 📌 Features

### 🤖 AI Chat Assistant
- Chat with a local AI model
- Ask questions on any topic
- Conversation history

### 📊 Dataset Analyzer
- Upload CSV datasets
- View statistical summary
- Interactive visualizations

### 🧠 AI Data Insights
- Generate insights from datasets using AI

### 📄 PDF Question Answering
- Upload PDF documents
- Ask questions about the content

### 💻 Code Generator
- Generate code in multiple languages
- Python, Java, C++, JavaScript

### 🗄 SQL Generator
- Convert natural language to SQL queries

### 📑 Resume Analyzer
- Analyze resumes using AI

### 🎯 ATS Resume Checker
- Get ATS score for job roles
- Identify missing keywords

### 📈 Dashboard
- Analytics of platform tools

---

## 🛠 Tech Stack

- Python

- Plotly
- PyPDF2
